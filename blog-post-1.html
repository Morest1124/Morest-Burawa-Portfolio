How I Built My Portfolio

A behind-the-scenes look at the design and code powering this site.

Building my portfolio was both a technical and creative journey. I started by outlining the core sections I wanted: About, Projects, Blog, and Contact. I chose HTML, CSS, and a bit of JavaScript for interactivity, keeping the design clean and responsive.

The biggest challenge was making the site both visually appealing and easy to navigate. I experimented with color palettes and typography until I found a style that felt personal yet professional.

For the blog, I created a simple grid layout to showcase posts. Each post links to a dedicated page for a full read, like this one. I also added a suggestion box so visitors can request topics.

Hosting was done via GitHub Pages, which made deployment seamless. I continue to update the site as I learn new skills and complete new projects.
